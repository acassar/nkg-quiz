# @nkg-quiz/shared-types

Package contenant tous les types TypeScript partagés pour l'application NKG Quiz.

## Installation

```bash
npm install @nkg-quiz/shared-types
```

## Utilisation

```typescript
import { Session, Question, Quiz, Category, Choice } from '@nkg-quiz/shared-types';
```

## Types disponibles

### Session
Types pour la gestion des sessions de quiz.

### Question  
Types pour les questions du quiz.

### Quiz
Types pour la structure des quiz.

### Category
Types pour les catégories de questions.

### Choice
Types pour les choix de réponses.

## Développement

```bash
npm run build    # Compiler les types
npm run dev      # Mode watch
```
